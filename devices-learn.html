Networking Devices — PrepPal

Common Devices & Purpose

R
Router
Connects networks, routes packets using IP. Router does NAT, routing protocols (OSPF, BGP).
S
Switch
Layer 2 device forwarding frames using MAC addresses — used in LANs.
H
Hub
Old broadcast device — sends to all ports (rare now).
AP
Access Point
Wireless device providing WiFi to clients; bridges wireless to wired LAN.
F
Firewall
Security appliance filtering traffic based on rules (stateful or stateless).

Placement Example

Design a small office network: internet → modem → router (NAT, DHCP) → switch → devices & AP. For interviews present IP plan and VLANs for separation.
