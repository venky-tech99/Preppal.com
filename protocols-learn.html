Key Protocols & Use-cases

HTTP/HTTPS
Use: Web traffic — HTTPS is encrypted (TLS).
Interview note: TLS handshake and certificate validation are common questions.
DNS
Use: Domain → IP resolution.
Explain iterative/recursive queries and record types (A/AAAA/CNAME).
DHCP
Use: Auto IP assignment.
DHCP states: DISCOVER → OFFER → REQUEST → ACK.
FTP / SFTP
Use: File transfer (SFTP over SSH is secure).
SMTP
Use: Sending email; often combined with IMAP/POP3 for retrieval.

Practical example: HTTPS request

Browser uses TLS handshake (ClientHello → ServerHello → certificates → key exchange) then HTTP encrypted data flows. Mention port 443 and certificate chain in interviews.
