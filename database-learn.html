What is a Database?

A Database is an organized collection of structured information, or data, typically stored electronically in a computer system. A database is controlled by a Database Management System (DBMS).

Key Concepts

1. Tables: Store data in rows and columns.

2. Primary Key: Unique identifier for each record in a table.

3. Foreign Key: Links two tables together.

4. Relationships: Define how tables connect with each other.

5. Normalization: Process to reduce redundancy and improve data integrity.

6. SQL vs NoSQL: SQL databases are relational, NoSQL are non-relational.

Example SQL Queries

CREATE TABLE Students (
  ID INT PRIMARY KEY,
  Name VARCHAR(50),
  Age INT
);

INSERT INTO Students VALUES (1, 'John Doe', 21);
SELECT * FROM Students;
