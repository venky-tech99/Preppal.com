Theory: Time, Speed & Distance

  • Formula: Speed = Distance / Time
  • Units: Convert all units to the same system (e.g., km/hr or m/s)
  • Conversions: 1 km/hr = (5/18) m/s and 1 m/s = (18/5) km/hr
  • Relative Speed: If objects move in opposite directions: add speeds; if in same direction: subtract
  • Average Speed: Total Distance / Total Time
  • Train Problems: Distance = Length of train; Time = Time taken to cross object/platform/person
  • Boats & Streams: Speed downstream = (u + v), upstream = (u - v)
