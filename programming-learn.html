What is Programming?

Programming is the process of creating instructions for a computer to perform specific tasks. It involves using a programming language to write code that can solve problems or automate processes.

Practice Questions

Q1: What will be the output of the following code?
int x = 5; x += 3; printf("%d", x);
Answer: 8 – Because x is increased by 3 using the += operator.

Q2: Which loop executes at least once?
Answer: do-while loop – It checks the condition after the loop body executes.

Q3: What is the time complexity of binary search?
Answer: O(log n) – Because it divides the search space in half each time.
